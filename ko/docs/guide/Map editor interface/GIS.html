GIS

  • GIS 데이터를 맵에 노출시킬 수 있다. 현재 불러온 GIS 데이터 목록이 뜬다. GIS 데이터 생성을 클릭하면 추가로 GIS 데이터를 불러올 수 있다.
  • GIS 데이터를 불러오면 아래와 같이 추가됨을 알 수 있다. 해당 데이터 오른쪽 설정 버튼을 클릭하여 좌표계 및 인코딩을 선택할 수 있다.

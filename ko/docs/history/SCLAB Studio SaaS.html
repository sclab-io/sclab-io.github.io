  • 컴바인 날짜에 시간, 분, 초 관련 기능 추가
  • 위치 아이콘 최대 사이즈 60px -> 720px 로 변경

버그 수정​

  • 에이전트 LIMIT 구문이 여러개인 경우 인식 갯수 관련 오류 수정
  • 맵 객체 로딩 오류 시 무한 로딩 오류 수정
  • 컴바인 아이템이 많은경우 브라우져가 꺼지는 오류 수정
  • S3 허브 커스텀 위치 아이콘 복사 버그 수정
